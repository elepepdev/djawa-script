Quick Start

Install JPL and run your first program in minutes.

1. Install via npm

Terminal
npm install -g @jawirhytam/djawascript

Or install the latest version from GitHub:

Terminal
npm install -g https://github.com/elepepdev/djawa-script

2. Create a new file

Terminal
djawa make hello

3. Write your first program (hello.jawa)

hello.jawa
iki iku name yoiku "World"
cetakno("Hello, " tambah name tambah "!")

4. Run it

Terminal
djawa hello.jawa

Code Blocks: terus & mbari

JPL is a block-based language. Every code block must open with terus and close with mbari.

lek (tenan) terus
  // code inside the block
mbari
Note: Use terus and mbari instead of curly braces {}.

Comments

// This is a single-line comment

/*
  This is a
  multi-line comment
*/

KeywordJS EquivalentDescription

Destructuring

Unpack values from objects or arrays into variables using terus mbari and [ ] syntax.

Object

jarno user yoiku terus name: "Budi", age: 30 mbari
jarno terus name, age mbari yoiku user
cetakno(name)  // Output: Budi
cetakno(age)   // Output: 30

Array

jarno arr yoiku [1, 2, 3]
jarno [a, b] yoiku arr
cetakno(a)  // Output: 1
cetakno(b)  // Output: 2

Default & Rename

jarno terus name, city dadi kutha mbari yoiku terus name: "Budi" mbari
cetakno(kutha)  // Output: undefined (no default)
jarno [x, y yoiku 5] yoiku [1]
cetakno(y)  // Output: 5

JPL also supports additional literal formats for numeric and string values.

Numeric Separator

Use underscore _ inside numeric literals to make large numbers more readable.

iki iku million yoiku 1_000_000
iki iku hex yoiku 0xFF_FF
cetakno(million)  // 1000000

Raw String

Add r prefix to strings to disable escape sequences. Backslashes are kept as literal characters.

iki iku path yoiku r"C:\windows\system32\user"
cetakno(path)  // Output: C:\windows\system32\user (no escape)

JPL also supports regular expression literals for pattern matching.

RegExp Literal

Write regular expressions directly with /.../flags syntax.

iki iku pattern yoiku /[0-9]+/g
iki iku emailPattern yoiku /^[^\s@]+@[^\s@]+$/
cetakno(emailPattern.tes("user@jawir.id"))  // Output: tenan (true)
cetakno(emailPattern.tes("invalid"))         // Output: salah (false)

Loops

kanggo — For

kanggo (jarno i yoiku 0 banjur i luwihCilik 5 banjur i yoiku i tambah 1) terus
  cetakno(i)
mbari
// Output: 0 1 2 3 4

selagi — While

jarno count yoiku 0
selagi (count luwihCilik 3) terus
  cetakno(count)
  count tambah 1
mbari

lakoni...selagi — Do...While

The block runs at least once, even if the condition is false from the start.

jarno count yoiku 0
lakoni terus
  cetakno("Count: " tambah count)
  count tambah 1
mbari selagi (count luwihCilik 3)

kanggo...ing — For...In

Iterates over the keys (property names) of an object.

iki iku user yoiku terus name: 'Sastro', age: 30, city: 'Jogja' mbari
kanggo (iki iku key ing user) terus
  cetakno(key tambah ": " tambah user[key])
mbari
// Output:
// name: Sastro
// age: 30
// city: Jogja

kanggo...soko — For...Of

Iterates over the values of an iterable (Array, String, Map, Set, etc.).

iki iku fruits yoiku ['apple', 'orange', 'mango']
kanggo (iki iku fruit soko fruits) terus
  cetakno(fruit)
mbari
// Output:
// apple
// orange
// mango

Pattern Matching: cocok

Branch on a value with structured patterns. Supports literals, bindings, wildcards, array patterns, and object patterns.

gawe classify(n) terus
  cocok n
    kalo 0 terus balekno "zero" mbari
    kalo 1, 2, 3 terus balekno "small" mbari
    kalo x lan x luwihCilik 0 terus balekno "negative: " + x mbari
    kalo [1, ...rest] terus balekno "starts with 1" mbari
    kalo terus name, age mbari terus balekno name + ", " + age mbari
    kalo _ terus balekno "other" mbari
  mbari
mbari
Note: The wildcard _ matches anything without binding.

KeywordJS EquivalentDescription

Arrow Functions: lakoni

Shorter function syntax, equivalent to JavaScript's => arrow functions.

  • Single-line: Implicit return — no balekno needed.
  • Multi-line: Wrap body in terus and mbari, use balekno explicitly.

Single-line (implicit return)

// JS: const multiply = (a, b) => a * b
iki iku multiply yoiku (a, b) lakoni a ping b
cetakno(multiply(7, 8))  // Output: 56

As a callback

iki iku numbers yoiku [1, 2, 3]
iki iku doubled yoiku numbers.petakake(n lakoni n ping 2)
cetakno(doubled)  // Output: [2, 4, 6]

Multi-line (explicit return)

iki iku greet yoiku (name) lakoni terus
  iki iku greeting yoiku "Welcome, " tambah name
  balekno greeting
mbari
cetakno(greet("Sastro"))  // Output: Welcome, Sastro

Rest & Spread

Use ... to collect remaining arguments (rest) or expand iterables (spread).

Rest parameters

gawe sum(...numbers) terus
  balekno numbers.kurangi((a, b) dadi a tambah b, 0)
mbari
cetakno(sum(1, 2, 3))  // Output: 6

Spread in arrays and function calls

jarno arr1 yoiku [1, 2, 3]
jarno arr2 yoiku [...arr1, 4, 5]
cetakno(arr2)  // Output: [1, 2, 3, 4, 5]

cetakno(sum(...arr1))  // Output: 6

Generator Functions

Use tenangan to declare a generator and asilno (yield) to pause and return values.

gawe tenangan idGenerator() terus
  jarno id yoiku 0
  selagi (tenan) terus
    asilno id tambah 1
  mbari
mbari

jarno gen yoiku idGenerator anyar()
cetakno(gen.next().value)  // Output: 1
cetakno(gen.next().value)  // Output: 2

asilno kabeh — Delegate

Use asilno kabeh to delegate to another generator (equivalent to yield*):

gawe tenangan inner() terus
  asilno 1
  asilno 2
mbari

gawe tenangan outer() terus
  asilno kabeh inner()
  asilno 3
mbari

jarno g yoiku outer anyar()
cetakno(g.next().value)  // Output: 1
cetakno(g.next().value)  // Output: 2
cetakno(g.next().value)  // Output: 3

Inline Assertion: pasten

Verify conditions and values inline. Failed assertions throw an error with a Javanese message.

pasten(1 tambah 1 plek 2)                                    // passes silently
pasten(1 tambah 1 plek 3, "basic math")                      // throws "pasten gagal: basic math"
pastenPodo([1, 2, 3], [1, 2, 3])                        // deep equality check
pastenPodo(actual, expected, "login must return token")  // custom message

Use these in test scripts or as runtime guards. pastenPodo is short for "pasten podo" (assert equals).

Optional Chaining: .mungkin.

Safely access nested properties without error if null/undefined. Equivalent to ?. in JavaScript.

iki iku user yoiku terus name: 'Sastro', address: terus street: 'Jl. Kenangan' mbari mbari
iki iku emptyUser yoiku kosong

// Safe access — works fine
iki iku street yoiku user.address.mungkin.street
cetakno(street)  // Output: Jl. Kenangan

// Safe access — property doesn't exist, returns undefined instead of crashing
iki iku postal yoiku user.address.mungkin.postalCode
cetakno(postal)  // Output: undefined

// Safe access on a null object
iki iku result yoiku emptyUser.mungkin.address.mungkin.street
cetakno(result)  // Output: undefined

Nullish Coalescing: utowoYenKosong

Returns the right-hand value only if the left-hand value is null/undefined — unlike utawa (||), it doesn't trigger for other falsy values like 0 or ''.

iki iku zeroValue yoiku 0
iki iku nullValue yoiku kosong
cetakno(zeroValue utowoYenKosong 10)    // Output: 0 (not null)
cetakno(nullValue utowoYenKosong 10)    // Output: 10 (null triggers fallback)

// Compare with logical OR (utawa)
cetakno(zeroValue utawa 10)  // Output: 10 (because 0 is falsy — different behavior!)

Tuple (Immutable Data)

A Tuple is like an array but its content cannot be changed after creation. Use it for data that should remain constant, like coordinates.

// Create a coordinate Tuple
iki iku coordinates yoiku tuple(10, 20)
cetakno(coordinates[0])  // Output: 10
cetakno(coordinates[1])  // Output: 20

// Attempting to change a Tuple will fail silently (or throw in strict mode)
cobak terus
  coordinates[0] yoiku 5
mbari nyekel (e) terus
  cetakno("Cannot modify a Tuple:", e.message)
mbari
cetakno(coordinates[0])  // Output: Still 10

// Tuples can hold mixed types
iki iku userInfo yoiku tuple("Budi", 28, tenan)
cetakno(infoUser[0])  // Output: Budi
cetakno(infoUser[1])  // Output: 28

Sleep / Wait: ngenteni

Pause program execution for a specified duration. Supports multiple time units for natural, conversational syntax.

SyntaxDescription
ngenteni(ms)Wait for ms milliseconds
ngenteni n detikWait for n seconds
ngenteni n menitWait for n minutes
ngenteni n jamWait for n hours
ngenteni n dinoWait for n days
 
cetakno("Starting...")
ngenteni(1000)           // wait 1000 milliseconds
cetakno("After 1 second")

ngenteni 3 detik         // wait 3 seconds
cetakno("After 3 seconds")

// With variables and expressions
jarno waktu yoiku 2
ngenteni waktu detik
ngenteni (1 tambah 1) detik

Note: ngenteni is async — it returns a Promise. Use it inside tenangan functions with enteni for sequential execution, or at top-level where the interpreter handles it automatically.

Constructs Still Requiring Parentheses

To keep the parser simple and unambiguous, parentheses are still required for:

  • Function declarations: gawe name (a, b) terus ... mbari
  • Arrow functions: (x, y) lakoni ...
  • C-style for: kanggo (i yoiku 0 banjur i luwihCilik 10 banjur i yoiku i tambah 1) terus ...
  • Function call expressions: f (a, b), tuple (a, b), takon (msg), anyar Foo (a, b)
  • Grouping expressions: (a tambah b)

Module System

JPL has a Python-like import system. You can import other .jawa files, npm packages, and built-in libraries.

JPL SyntaxPython EquivalentDescription
jupukno nameimport nameSimple import (auto-search)
jupukno name dadi aliasimport name as aliasImport with alias
jupukno name soko 'path'from path import nameFrom-import
jupukno name dadi alias soko 'path'from path import name as aliasFrom-import with alias
jupukno * soko 'path'from path import *Wildcard import

Both soko and saka work interchangeably.

Module search order: relative path → current dir → lib/ → bundled stdlib → node_modules/

util.jawa

gawe greet(name) terus
  balekno "Hello, " tambah name
mbari

iki iku VERSION yoiku "1.0"

metokno terus greet mbari          // Named export
metokno biasane VERSION    // Default export

app.jawa

jupukno greet, VERSION saka './util.jawa'

cetakno("Version:", VERSION)   // Output: Version: 1.0
cetakno(greet("Doni"))         // Output: Hello, Doni

You can also import npm packages directly:

jupukno lodash           // import lodash from node_modules
jupukno express          // import express from node_modules
