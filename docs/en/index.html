Programming Language
Based on Javanese

JPL (Javanese Programming Language) is an interpretative programming language with Javanese syntax. Write code in your native tongue, build powerful programs.

Start Learning GitHub

Why JPL?

Javanese-Based Syntax

Keywords in Javanese Ngoko. Write code daily using the language you know.

Easy to Learn

If you're familiar with JavaScript, JPL will feel natural. Same concepts, different language.

Powerful Pattern Matching

A robust pattern matching system to handle complex data elegantly.

Full OOP Support

Classes, interfaces, enums, structs, abstract classes, and sealed classes — all available.

Rich Standard Library

Math, Date, JSON, Array methods, String methods, and more — ready to use.

Powerful CLI

Run, format, and create new files directly from the command line with intuitive commands.
