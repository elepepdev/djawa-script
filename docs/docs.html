Mulai Cepat

Install JPL dan jalankan program pertama Anda dalam hitungan menit.

Terminal 
# Install via npm
npm install -g @jawirhytam/jawirscript

# Buat file baru
djawa make halo

# Jalankan
djawa halo.jawa

Program Pertama

halo.jawa 
iki iku jeneng yoiku "Dunia"
cetakno("Halo, " tambah jeneng tambah "!")

Blok Kode: terus & mbari

JPL adalah bahasa berbasis blok. Setiap blok kode harus dibuka dengan terus dan ditutup dengan mbari.

contoh.jawa 
lek (tenan) terus
  // kode di dalam blok
mbari
Catatan: Gunakan terus dan mbari bukan kurung kurawal {}. Error akan muncul jika Anda mencoba menggunakan { atau }.

Variabel

Kata Kunci Ekuivalen JS Keterangan

// Variabel yang bisa diubah

Pattern Matching: cocok

gawe klasifikasi(n) terus
  cocok n
    kalo 0 terus balekno "nol" mbari
    kalo 1, 2, 3 terus balekno "siji-siji" mbari
    kalo x lan x luwihCilik 0 terus balekno "negatif: " + x mbari
    kalo x terus balekno "liyane: " + x mbari
  mbari
