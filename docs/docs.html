Mulai Cepat

Install JPL dan jalankan program pertama Anda dalam hitungan menit.

1. Install via npm

Terminal
npm install -g @jawirhytam/jawirscript

Atau install versi terbaru dari GitHub:

Terminal
npm install -g https://github.com/elepepdev/djawa-script

2. Buat file baru

Terminal
djawa make halo

3. Tulis program pertama Anda (halo.jawa)

halo.jawa
iki iku jeneng yoiku "Dunia"
cetakno("Halo, " tambah jeneng tambah "!")

4. Jalankan

Terminal
djawa halo.jawa

Blok Kode: terus & mbari

JPL adalah bahasa berbasis blok. Setiap blok kode harus dibuka dengan terus dan ditutup dengan mbari.

mbari
Catatan: Gunakan terus dan mbari bukan kurung kurawal {}.

JPL juga mendukung format literal tambahan untuk nilai numerik dan string.

Numeric Separator

Gunakan garis bawah _ di dalam literal numerik supaya angka besar lebih mudah dibaca.

iki iku juta yoiku 1_000_000
iki iku hex yoiku 0xFF_FF
cetakno(juta)  // 1000000

Raw String

Tambahkan awalan r pada string untuk menonaktifkan escape sequence. Backslash tetap sebagai karakter literal.

iki iku path yoiku r"C:\windows\system32\user"
cetakno(path)  // Output: C:\windows\system32\user (tanpa escape)

JPL juga mendukung literal regular expression untuk pencocokan pola.

RegExp Literal

Tulis regular expression langsung dengan sintaks /.../flags.

iki iku pola yoiku /[0-9]+/g
iki iku emailPattern yoiku /^[^\s@]+@[^\s@]+$/
cetakno(emailPattern.tes("user@jawir.id"))  // Output: tenan (true)
cetakno(emailPattern.tes("ora-valid"))      // Output: salah (false)

Pattern Matching: cocok

Bercabang pada sebuah nilai dengan pola terstruktur. Mendukung nilai literal, binding, wildcard, pola array, dan pola objek.

Arrow Function: lakoni

Sintaks fungsi yang lebih pendek, ekuivalen dengan arrow function => di JavaScript.

  • Satu baris: Return implisit — tidak perlu balekno.
  • Multi-baris: Bungkus badan dalam terus dan mbari, gunakan balekno secara eksplisit.

Satu baris (return implisit)

// JS: const multiply = (a, b) => a * b
iki iku kalikan yoiku (a, b) lakoni a ping b
cetakno(kalikan(7, 8))  // Output: 56

Sebagai callback

iki iku angka yoiku [1, 2, 3]
iki iku ganda yoiku angka.petakake(n lakoni n ping 2)
cetakno(ganda)  // Output: [2, 4, 6]

Multi-baris (return eksplisit)

iki iku sapa yoiku (jeneng) lakoni terus
  iki iku sapaan yoiku "Sugeng rawuh, " tambah jeneng
  balekno sapaan
mbari
cetakno(sapa("Sastro"))  // Output: Sugeng rawuh, Sastro

Rest & Spread

Gunakan ... untuk mengumpulkan sisa argumen (rest) atau memperluas iterable (spread).

Rest parameters

gawe jumlah(...angka) terus

mbari
cetakno(jumlah(1, 2, 3))  // Output: 6

Spread dalam array dan pemanggilan fungsi

jarno arr1 yoiku [1, 2, 3]
jarno arr2 yoiku [...arr1, 4, 5]
cetakno(arr2)  // Output: [1, 2, 3, 4, 5]

cetakno(jumlah(...arr1))  // Output: 6

Optional Chaining: .mungkin.

Akses properti yang bersarang dengan aman tanpa error jika null/undefined. Ekuivalen dengan operator ?. di JavaScript.

// Akses aman pada objek null
iki iku result yoiku emptyUser.mungkin.alamat.mungkin.jalan
cetakno(result)  // Output: undefined

Nullish Coalescing: utowoYenKosong

Mengembalikan nilai kanan hanya jika kiri null/undefined — berbeda dengan utawa (||), tidak aktif untuk nilai falsy lain seperti 0 atau ''.

iki iku nilaiNol yoiku 0
iki iku nilaiKosong yoiku kosong
cetakno(nilaiNol utowoYenKosong 10)    // Output: 0 (bukan null)
cetakno(nilaiKosong utowoYenKosong 10) // Output: 10 (null memicu fallback)

// Bandingkan dengan logical OR (utawa)
cetakno(nilaiNol utawa 10)  // Output: 10 (karena 0 adalah falsy — perilaku berbeda!)

Tuple (Data Immutable)

Tuple seperti array tapi tidak bisa diubah setelah dibuat. Gunakan untuk data yang harus tetap konstan, seperti koordinat atau pasangan tetap.

// Buat Tuple koordinat
iki iku koordinat yoiku tuple(10, 20)
cetakno(koordinat[0])  // Output: 10
cetakno(koordinat[1])  // Output: 20

// Mencoba mengubah Tuple akan gagal secara senyap (atau throw di strict mode)
cobak terus
  koordinat[0] yoiku 5
mbari nyekel (e) terus
  cetakno("Cannot modify a Tuple:", e.message)

cetakno(koordinat[0])  // Output: Masih 10

// Tuple bisa menampung tipe campuran
iki iku infoUser yoiku tuple("Budi", 28, tenan)
cetakno(infoUser[0])  // Output: Budi
cetakno(infoUser[1])  // Output: 28

Tanda Kurung Opsional

Mengikuti filosofi "berbicara langsung ke komputer", tanda kurung () bersifat opsional pada beberapa konstruk pada level pernyataan.

Konstruk yang Masih Membutuhkan Kurung

Untuk menjaga parser tetap sederhana dan tidak ambigu, kurung tetap wajib untuk:

  • Deklarasi fungsi: gawe nama (a, b) terus ... mbari (multi-param, destructuring)
  • Arrow function: (x, y) lakoni ...
  • C-style for: kanggo (i yoiku 0 banjur i luwihCilik 10 banjur i yoiku i tambah 1) terus ...
  • Ekspresi pemanggilan fungsi: f (a, b), tuple (a, b), takon (msg), anyar Foo (a, b)
  • Grup ekspresi: (a tambah b)
