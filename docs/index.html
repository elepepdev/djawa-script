Bahasa Pemrograman
Berbasis Jawa

JPL (Jawa Programming Language) adalah bahasa pemrograman interpretatif dengan sintaks berbahasa Jawa. Tulis kode dalam bahasa ibu, hasilkan program yang powerful.

Mulai Belajar GitHub

Kenapa JPL?

Sintaks Berbasis Jawa

Kata kunci dalam bahasa Jawa ngoko. Tulis kode sehari-hari dengan bahasa yang Anda kenal.

Mudah Dipelajari

Jika Anda familiar dengan JavaScript, JPL akan terasa natural. Konsep yang sama, bahasa yang berbeda.

Pattern Matching

Sistem pattern matching yang kuat untuk menangani data kompleks dengan elegan.

OOP Lengkap

Kelas, interface, enum, struct, abstract class, dan sealed class — semuanya tersedia.

Standard Library Kaya

Math, Date, JSON, Array methods, String methods, dan banyak lagi — siap digunakan.

CLI Powerful

Jalankan, format, dan buat file baru langsung dari command line dengan perintah intuitif.
